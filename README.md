
**HHS Stroke Protocol Engine**
- A streamlined application designed to manage and execute stroke protocol workflows.

**Features**
- Dynamic Protocol Management: Easily add, edit, and organize protocol steps.

**User-Friendly Interface:** Clean, card-based navigation for quick access.

**Offline Capability:** Fully functional standalone desktop application.

**Auto-Update Support**: Built-in support for seamless updates as new protocol versions are released.

**Installation**
- Navigate to the Releases page.
- Download the latest HHS Stroke Protocol Engine Setup file (.exe).
- Run the installer and follow the on-screen prompts.
- Launch the app from your Desktop or Start Menu.

Usage
- Adding Steps: Use the interface to input new protocol steps.

**Editing**: Modify existing criteria using the "Edit" function on each card.

**Data Sync**: All changes are automatically saved locally upon committing.

**Disclaimer**
**Note: This application is a tool for organizing protocols and is intended for informational/procedural support. Always ensure protocol updates are verified against current medical standards and institutional guidelines before clinical application.**
